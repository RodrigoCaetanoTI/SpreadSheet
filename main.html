Fully functional spreadsheet that can be used on a web page to enter or display data. The only thing you need is a div:
<div id = 'mydiv'></div>
var mySpreadsheet = $( '#mydiv' ).spreadsheet( options );
It's that easy.
  • Resizable columns
  • Excel-style headers that user or programmer can hide
  • Can be displayed in zebra-striped mode for better presentation
  • Data can be passed to initialize the spreadsheet
  • User or programmer can add and delete rows and columns
  • Unlike other spreadsheets, display only as many rows and columns as you need. Add or delete them in your script as you need, or allow a user to do so.
  • Can be initialized to display-only mode
  • A user can be allowed to see a context menu to customize the spreadsheet
  • Any column is sortable, using string or number sort, depending on the data
  • Data can be deleted, copied, and pasted, from / to a single cell or an entire row, inside a spreadsheet or between multiple spreadsheets
  • Keyboard interaction (Ctrl+C, Ctrl+V, Enter, and Tab / Shift+Tab keys)
  • The data can be retrieved into a two-dimensional array of strings
  • Multiple spreadsheets can be displayed on a page
  • Fully exposed API
Double-click on a cell to enter data.
Click on headers to select a row or a column.
Use right mouse-click to open a context menu and customize the spreadsheet.
